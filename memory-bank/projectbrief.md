# Project Brief: CriticalMinds («Оптика»)

## Что это
Мобильное/веб-приложение для развития критического мышления и осознанности у взрослых. Рабочее название прототипа — «Оптика» (прибор для наблюдения за собственным мышлением).

## Зачем
Заменяет преподавателя: теория и индивидуальные практики доставляются через самостоятельные микроуроки в приложении, а живое обучение происходит на еженедельных групповых встречах без ведущего-эксперта (ротируемый фасилитатор по сценарным карточкам из приложения).

## Ключевые требования
1. **Программа:** 12 недель, 4 модуля, 36 микроуроков по 10–15 минут (полная карта: `docs/karta_36_urokov.md`).
2. **Формат урока (неизменный):** Крючок (пользователь сначала ошибается сам) → Концепт (строго один) → Проверка → Перенос в жизнь.
3. **Сквозные инструменты:** дневник решений, трекер прогнозов, еженедельный калибровочный квиз, график калибровки как главный экран.
4. **Групповой слой:** сценарные карточки встреч для ротируемого ведущего (`docs/scenarnye_kartochki_vstrech.md`), подготовка «что принести на встречу».
5. **Кураторский раздел:** метаданные активности всех участников (когда/как быстро пройдены уроки, счётчики практик, автофлаги внимания). **Железный принцип: куратор видит количество, но не содержание** — тексты дневников и формулировки прогнозов приватны.

## Целевая аудитория
Первая группа: 5–7 руководителей ИТ-кластера МТС. Куратор — Андрей (директор, он же участник).

## Границы (не делаем)
- Не LMS общего назначения, не платформа курсов — одна программа, глубоко.
- Никакого доступа куратора к содержанию личных записей — ни в какой версии.
- Без геймификации ради геймификации: единственная соревновательная механика — калибровочный турнир прогнозов.

## Критерий успеха продукта
Через 12 недель группа проходит слепое сравнение «до/после» (разборы недели 1 vs недели 12) с видимым прогрессом, а личные графики калибровки участников приближаются к диагонали.
