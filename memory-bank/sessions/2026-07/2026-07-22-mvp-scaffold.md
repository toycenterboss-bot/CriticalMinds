# Сессия 2026-07-22: от репозитория до живого MVP

## Хронология
1. **Подключение репозитория.** Локальная папка полная, GitHub пуст. Пуш из облачной песочницы Cowork заблокирован прокси; первый fine-grained токен оказался read-only. Решение: git на Mac через Desktop Commander + classic-токен. Коммит 3bb0a60 — весь проект на GitHub.
2. **Архитектурный план** (docs/architecture.md): по AskUserQuestion выбраны Postgres в Docker, инвайт-ссылки, план+скаффолд сразу. Роли superadmin/curator/participant, приватность на уровне API-схем.
3. **Каркас** (56747a6): FastAPI (модели, JWT+argon2, инвайты, RBAC, изоляция групп 404), тесты приватности/изоляции, Vite-фронт (логин, /join, ролевой роутинг), docker-compose, README. E2E в облаке на sqlite: секретные тексты в кураторский дашборд не утекают.
4. **Запуск на Mac.** Грабли: порты 5432/5433 заняты (локальный Postgres, InvestHub) → 15433 (1533fc4); NODE_ENV=production глобально → npm без dev-зависимостей («vite: command not found»). Стенд поднят, логин суперадмина проверен.
5. **Порт движка уроков** (354fbe5): LessonPlayer (6 типов шагов), дневник, прогнозы, квиз с разбором ответов (расширение API), график калибровки, кураторский дашборд в стиле прототипа. Палитра C выровнена по прототипу 1:1. Seed: полный контент недели 1.
6. **Первые UX-правки Андрея** (73d23ea): явные метки верно/мимо в проверках; началась сага о расцветке (см. сессию 23.07).
