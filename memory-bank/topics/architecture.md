# Topic: Архитектура и стек
> Заведён 2026-07-23 (харвест techContext + systemPatterns). Подробный документ: ../../docs/architecture.md

## Состояние
Работает на Mac Андрея: Postgres 16 в Docker (порт 15433), FastAPI на :8000, Vite dev на :5173. Все тесты зелёные. Прод-хостинга нет.

## Стек и почему
- **PostgreSQL 16 (Docker)** — настоящая СУБД с первого дня (требование Андрея), та же в прод; JSONB для контента.
- **FastAPI + SQLAlchemy 2.0 + psycopg** — знаком Андрею; Pydantic-схемы = естественный механизм приватности.
- **JWT**: access 30 мин в памяти SPA + refresh HttpOnly cookie с ротацией и хэшами в БД; пароли argon2id; rate limit на /auth/* (slowapi).
- **React 18 + Vite**, инлайн-стили + токены; без CSS-фреймворков.
- Онбординг без SMTP: одноразовые инвайт-ссылки (SHA-256 хэш в БД, TTL 72 ч).

## Ключевые паттерны
- **Урок = данные + движок**: `lessons.steps` JSONB, 7 типов шагов (hookNumber, hookChoice, hookText, concept, check, firstEntry, transfer). Движок — `LessonPlayer.jsx`. Контент 36 уроков не потребовал нового кода, кроме hookText.
- **Гейтинг v2**: неделя N+1 открывается после недели N (все уроки + квиз). Календарный гейтинг убран. Логика: `me.py::week_statuses`. Карточки встреч НЕ гейтятся (ведущему готовиться вперёд).
- **Калибровка сквозная**: любая механика с процентами уверенности обязана попадать на график (`/me/calibration`: квизы при 90% + бакеты прогнозов).
- **Изоляция групп**: `auth.py::get_own_group`; чужой куратор получает 404 (не 403) — не раскрываем существование группы.
- **Автофлаги** — чистая функция над метаданными (`flags.py::team_flags`), 5 правил; новые — туда же.
- **Валидация текстов** централизована: `schemas.clean_text` (длины, минимум букв, разнообразие, вычистка невидимых символов).
- **Групповые артефакты**: журнал встреч (meeting_logs) — общий документ (группа + куратор), уникальность (group, week). Личное остаётся личным.
- **Seed гранулярный**: каждый тип контента заливается при пустой таблице; новый тип не требует refresh и не сбрасывает прогресс. `python -m app.seed refresh` — принудительная перезаливка (сносит прогресс уроков/квизов).

## Грабли
- ⚠️ EmailStr отвергает домены *.local — для dev использовать *.app / example.com (симптом: 422 на логине сида).
- ⚠️ PredictionOut наследовал PredictionIn → выходные данные прогонялись через входные валидаторы и падали на старых записях. Out-схемы не наследовать от In.
- ⚠️ Кавычки « „ " внутри python-строк контента ломали парсер (SyntaxError на '»') — внутри строк только «ёлочки», без „лапок".
- ⚠️ Alembic-миграций нет: create_all добавляет таблицы, но не меняет колонки. Ввести ДО реальной группы.
