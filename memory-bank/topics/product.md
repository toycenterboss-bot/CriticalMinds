## Что строим и зачем
Приложение-тренажёр критического мышления для взрослых. Заменяет преподавателя: теория и личные практики — микроуроками в приложении, живое обучение — на еженедельных встречах без ведущего-эксперта (ротируемый фасилитатор по сценарным карточкам). Программа: 12 недель, 4 модуля, 36 уроков по ~10 минут.

Ключевые требования: формат урока неизменен (Крючок → один Концепт → Проверка → Перенос); сквозные инструменты — дневник решений, трекер прогнозов, еженедельный калибровочный квиз, график калибровки как главный экран; групповой слой — сценарные карточки встреч + журнал артефактов; кураторский раздел — только метаданные.

## Принципы (проверять каждую фичу)
- Куратор видит количество, но не содержание — уровень модели данных/API, не UI. Распространяется и на superadmin, и на самоотчёты (барометр: только агрегат при ≥3).
- Уроки = данные для движка, не код (новый тип шага — только для принципиально нового формата).
- Мотивация без токсичности: себя участник видит явно, других — анонимно/агрегированно; адресного «вы отстали» нет. Хвалим первых, молчим про последних.
- Стрик неделями, не днями — взрослая аудитория.
- Язык продукта и документации — русский.

## Границы (не делаем)
Не LMS общего назначения — одна программа, глубоко. Никакого доступа куратора к содержанию личных записей — ни в какой версии. Без геймификации ради геймификации (набор механик расширен явным решением Андрея 23.07: вехи, пульс, первопроходец, турнир).
