## Роли цвета (выстрадано тремя итерациями с Андреем)
- **Цвет результата — всегда про решение пользователя**, не про «правильность варианта вообще»: зелёный = ты был прав, красный = ошибся.
- Яркая заливка только у ДЕЙСТВИЙ и ОШИБОК: выбран верно (teal), выбран зря / пропущен верный (red). Правильно проигнорированные — нейтральные с тихой пометкой «не выбран ✓». Иначе «зелёный хор» топит суть.
- Вердикт всегда явный: «ваш ответ: X ✓ / ✗ · верно: Y», отдельной строкой под вариантом (float-right сжимался).
- red — ошибки участника и risk-флаги; marker — внимание/крючки/бренд-акцент.

## Фирменные жесты
Тёмная «приборная» шапка (ink + теал-кант + «ОПТИКА_» с маркерным курсором); маркерные выделения заголовков (.hl — жёлтый штрих); hero с прогресс-кольцом недели; карта 12 недель; line-иконки в нижней навигации; fade-up на вкладках/шагах; focus-ring теал; prefers-reduced-motion учтён.
